Bash Configurations
===================

Usage
-----

### Mac and Linux ###

```shell
cd ~
ln -s $DOTFILES_HOME/bash/bash_include .bash_include
ln -s $DOTFILES_HOME/bash/bash_profile .bash_profile
ln -s $DOTFILES_HOME/bash/bashrc .bashrc
```
